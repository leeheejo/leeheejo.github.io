오랜만에 알고리즘 문제를 풀다가 
문자열이 특정 문자열로 시작하는지 검사해야했는데 이 메소드가 생각이 안났다. 
기억해두자.

```
String s = "start";
String s2 = "startsWith";

s.startsWith(s2); //true
```
